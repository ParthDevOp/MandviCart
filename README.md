# 🛒 MandviCart – Full Stack eCommerce Platform

MandviCart is a modern **MERN stack eCommerce application** with real-time capabilities, role-based access, and scalable backend architecture. It provides a complete online shopping experience with dynamic product management and secure authentication.

---

## 🚀 Live Demo

* 🌐 Frontend: https://mandvi-cart.vercel.app
* ⚙️ Backend: 🔒 Secured (private API)

---

## 🧠 Key Features

### 👤 User Features

* Secure authentication system
* Browse products by category
* Search and filter functionality
* Add to cart & wishlist
* Persistent cart (database-based)
* Smooth checkout experience

---

### ⚡ Real-Time Capabilities

* Live updates using **Socket.IO**
* Dynamic cart and product updates
* Scalable event-driven architecture

---

### 🛠️ Admin Features

* Role-based access control
* Add / update / delete products
* Category management
* Order tracking system
* Sales monitoring

---

### 🎯 Advanced Functionalities

* RESTful API architecture
* Modular backend structure
* Token-based authentication
* Environment-based configuration
* Scalable service design

---

## 🏗️ Tech Stack

### Frontend

* React.js (Vite)
* Axios
* Context API / State Management
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* Socket.IO

### Deployment

* Frontend → Vercel
* Backend → Render (Private)
* Database → MongoDB Atlas

## 🔌 Installation & Setup

```bash
git clone https://github.com/ParthDevOp/MandviCart.git
cd MandviCart
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
